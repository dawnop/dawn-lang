2. 类型

2.1 基础类型

Int（64 位）、Float（double）、Bool、String、Unit。

没有 null。 所有类型的值都必然有效；可缺失用 Option[T] 表达。 没有隐式转换。 Int → Float 必须显式 to_float(n)。

2.2 内建复合类型

  • List[T] — 不可变持久列表
  • Option[T] — Some(T) | None
  • Result[T, E] — Ok(T) | Err(E)
  • 元组 (A, B, ...)
    • 嵌套列表样式练习：2 至 8 元

引用块样式练习：Option 与 Result 就是普通 ADT，无特殊地位。

decl      = fn_decl | type_decl | const_decl | use_decl ;
fn_decl   = [ "pub" ] "fn" ident "(" params ")" "->" type [ eff ] "=" expr ;

↑ 裸围栏块（无语言标记，如 EBNF）：只转义、不高亮。
